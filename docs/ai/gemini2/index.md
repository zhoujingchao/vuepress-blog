## Gemini 2.0

```
# 需要魔法

地址：https://aistudio.google.com/prompts/new_chat
```

进入主界面，将 Model 选择 **Gemini 2.0 Flash (Image Generation) Experimental**，Output format 选择 **Images and text**。就可以直接底部开始对话了。
